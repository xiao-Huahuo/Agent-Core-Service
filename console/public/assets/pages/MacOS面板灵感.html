ABOUT.md

## 为什么选择 Agent Skills Marketplace？

在数以千计的 GitHub 仓库中找到合适的 agent skill 可能让人不知所措。Skills Marketplace 通过提供智能搜索、职业筛选和质量指标来解决这个问题，帮助您快速找到所需的技能。

无论您是自动化工作流的开发者、构建自定义 AI 工具的团队负责人，还是探索 AI 编程助手的爱好者，您都能找到适合各种用例的技能。所有技能采用开放的 SKILL.md 标准，兼容 Claude Code、OpenAI Codex CLI 等工具。

# 常见问题

Claude Code、OpenAI Codex 及 AI 编程工具的 Agent Skills 指南 — 发现、安装和创建自定义技能（2026）

/**
* Agent Skills 是扩展 AI 编程助手功能的模块化能力。每个 skill 由一个包含指令的 SKILL.md 文件以及可选的脚本和模板组成。2025 年 12 月，Anthropic 将 Agent Skills 规范作为开放标准发布，OpenAI 也在 Codex CLI 和 ChatGPT 中采用了相同格式。Skills 由模型调用——AI 会根据上下文自主决定何时使用它们。
*/
/**
* Claude Code：添加到 ~/.claude/skills/（个人）或 .claude/skills/（项目）。OpenAI Codex CLI：添加到 ~/.codex/skills/。两者使用相同的 SKILL.md 格式。克隆 GitHub 仓库并将 skill 文件夹复制到对应目录，AI 会自动发现并加载。
*/
/**
* 本市场的技能来自公共 GitHub 仓库。我们会过滤掉低质量的仓库（最少 2 星）并扫描基本质量指标，但你应该在安装前始终审查代码。社区技能应像对待任何开源代码一样——使用前先检查。
*/
/**
* 可以! Claude Code 支持同时使用多个技能。技能是模块化的,设计为可以协同工作。例如,你可以将「代码审查」技能与「Git 自动化」技能结合使用,以自动审查并提交代码更改。Claude 会根据你的请求上下文智能选择合适的技能。
*/
/**
* 可以! 你可以创建自定义 agent skill 并在 GitHub 上分享。查看技能创建器 (skill-creator) 技能获取创建自己技能的详细指导。基本结构需要一个包含指令的 SKILL.md 文件。你还可以添加可选的脚本和模板。
*/
/**
* 技能由模型调用——Claude 会根据上下文自动决定何时使用它们。斜杠命令由用户调用——你需要明确输入命令来触发它们。技能使你的工作流程能够实现更智能、更具上下文感知的自动化。
*/
/**
* 我们的爬虫定期与 GitHub 同步以获取最新的技能和更新。当技能仓库在 GitHub 上更新时,更改会在下次同步后不久出现在 SkillsMP 上。你可以在每个技能卡片上看到最后更新时间戳。我们建议定期查看新技能和现有技能的改进。
*/
/**
* 没有，SkillsMP 是一个独立的社区项目。我们从 GitHub 聚合和展示 agent skills 以便于发现。官方 skills 和文档请参考 Anthropic 的 github.com/anthropics/skills 或 OpenAI 的 Codex 文档。
*/
